# NewTube

NewTube is a clone of Youtube.

## Tech Stack

- Next.js
- TypeScript
- Bun
- Tailwind CSS
- Shadcn UI
- Clerk
- Drizzle ORM
- Neon DB with Postgres SQL
- trpc
- Upstash with Redis, Workflow and Ratelimit
- Mux
- Uploadthing
- OpenAI
