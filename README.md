# NewTube

NewTube is a clone of Youtube.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI
- Clerk
- Drizzle ORM
- Neon DB with Postgres SQL
- trpc
- Upstash with Redis
